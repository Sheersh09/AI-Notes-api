# AI Notes API

A learning project built with FastAPI to understand backend API development before building a full AI Study Partner application.

## Goals

- Learn FastAPI fundamentals
- Build REST APIs
- Practice SQLAlchemy and SQLite
- Integrate an LLM (Gemini/OpenRouter)
- Deploy using Render

## Run Locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
